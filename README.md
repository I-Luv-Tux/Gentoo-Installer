# Gentoo-Installer

  This is a lightweight gentoo install script with support for optical and nvme storage devices

  i also made the whole script 1 line with no comments for fun

## What to Expect  
A basic openrc install  
A working swap partition  
A cron daemon  
A logging daemon  
An almost untouched make.conf  
Setup CPU flags but video card is not setup  
A working PARTUUID based fstab file  
NetworkManager  
Nvme IO scheduling  
GRUB bootloader
